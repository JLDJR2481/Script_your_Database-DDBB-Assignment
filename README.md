## Scripts
### ddl.sql
Este script recoge los comandos que son necesarios para poder crear la base de datos Restaurant, que recoge las tablas Employee, Orders y Restaurant

### dml.sql
Este script recoge los comandos que hemos usado para insertar los datos en las tablas de nuestra base de datos

### queries.sql
Este script se utiliza para hacer algunos queries básicos para interactuar con la base de datos

### delete.sql
Este script se utiliza para borrar todos los datos que existan en las tablas de la base de datos.

### drop.sql
Este script permitirá hacer un drop a la base de datos eliminandola
